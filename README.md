# Appium test

The project can be run using the docker-compose. But don't forget to turn on the appium on your test machine locally.
Project was tested on such Android emulator  - **Pixel_3a_API_34_extension_level_7_arm64-v8a**.
Please install trapadvisor apk file before testing.

Fast API supports automatic documentation generation so please take a look at /docs endpoint.
